# Carter's World 🎮

A 3D driving-and-snacks game for Carter. **Play: https://jjmzeus36.github.io/carters-world/**

## Develop (any machine)
```bash
git clone https://github.com/jjmzeus36/carters-world.git
cd carters-world
# edit files in src/ , then:
bash build.sh
python3 -m http.server 8123   # test at http://localhost:8123
git add -A && git commit -m "..." && git push   # deploys to the live link in ~1 min
```
See `CLAUDE.md` for the full project guide, design rules, and gotchas.
